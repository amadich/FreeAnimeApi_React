#
Animation Navigation Texture!
```js
// animataion texteur !
     setTimeout(() => {
         let t = document.getElementById("title_nav");
         let text = 'Anime-Images API'
         let i = 0;
         let type = () => {
         if (i < text.length) {
               t.innerHTML += text.charAt(i);
               i++;
               setTimeout(type, 100)
         }
  
         }
         type();
      },500)

  
      // fin animation
```
